
# Bienvenue

Ceci est le wiki de l'équipe d'administration système et réseau de FDN.
Hormis cette page d'accueil, le wiki nécessite une authentification avec les
mots de passe adhacc-XXX, que chaque membre reçoit lors de son adhésion.  Ce
wiki est accessible en lecture pour l'ensemble des membres ; seul·e·s les membres de
l'équipe adminsys peuvent le modifier.

# Nous rejoindre

Si vous souhaitez nous rejoindre, les meilleurs moments sont les rencontres
physiques : admincamp, assemblée générale. Vous trouverez l'ensemble des infos
utiles sur la page de l'[équipe](https://git.fdn.fr/adminsys/wiki/-/tree/vlp_update_01/equipe).

# Camps

Ces rencontres sont ouvertes à toute personne membre qui souhaite s'investir dans le fonctionnement des services et du réseau de FDN.

## Le dernier :

Pour se rappeler le bon vieux temps, voici les infos du derniers [Camp physique](https://www.fdn.fr/fdncamp-des-22-23-fevrier-2020/) !

## Le prochains :

Pour les retardataires, nous subissons une pandémie mondiale. Les Camps restent pour le moment 100% remote. On vous tient au courant pour les dates sous peu !
